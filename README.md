## Project: DNA Sequence Toolkit using Python

This project contains beginner-friendly Python scripts for basic DNA sequence analysis.

## Files Included

| File | Description |
|---|---|
| `python-basics/dna_reverse_complement.py` | Calculates the reverse complement of a DNA sequence |
| `python-basics/gc_content_calculator.py` | Calculates GC content percentage of a DNA sequence |
| `python-basics/dna_to_rna.py` | Converts a DNA sequence into an RNA sequence |
| `python-basics/motif_finder.py` | Finds positions of a motif in a DNA sequence |
| `python-basics/kmer_counter.py` | Counts k-mers in a DNA sequence |
