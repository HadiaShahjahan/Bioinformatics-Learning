# BLAST Mini Project

This project demonstrates a beginner-level sequence similarity search using NCBI BLAST.

## Project Aim

The aim of this project is to use BLAST to compare a DNA query sequence against sequence databases and understand the top matching results.

## Tool Used

- NCBI BLAST
- Nucleotide BLAST (BLASTn)

## Files Included

| File | Description |
|---|---|
| `query_sequence.txt` | DNA sequence used as the BLAST query |
| `blast_results_summary.md` | Summary and interpretation of BLAST results |

## Skills Practiced

- Using NCBI BLAST
- Working with FASTA-formatted DNA sequences
- Understanding sequence similarity
- Reading BLAST result tables
- Interpreting percent identity, E-value, and query cover

## Learning Outcome

This project helped me understand how bioinformatics tools can be used to identify similar DNA sequences and interpret biological database search results.
