
# FASTQ Quality Analysis using Python

This project demonstrates a beginner-level analysis of FASTQ sequencing reads using Python.

## Project Aim

The aim of this project is to understand the structure of FASTQ files and calculate basic sequencing read quality statistics using Python.

## Background

FASTQ files store sequencing reads along with quality scores. Quality analysis is an important first step before downstream bioinformatics workflows such as alignment, variant calling, and RNA-seq analysis.

A small example FASTQ file was used for practice and demonstration purposes.

## Files Included

| File | Description |
|---|---|
| `sample_reads.fastq` | Small example FASTQ file used for practice |
| `fastq_quality_analysis.py` | Python script for analyzing read count, read length, and quality scores |
| `results_summary.md` | Summary of the analysis output and interpretation |

## Skills Practiced

- Reading FASTQ files
- Understanding sequencing reads
- Working with quality score characters
- Python file handling
- Loops and lists
- Basic sequencing quality interpretation

## Learning Outcome

This project helped me understand how raw sequencing reads are stored in FASTQ format and how Python can be used to calculate simple read quality statistics.
