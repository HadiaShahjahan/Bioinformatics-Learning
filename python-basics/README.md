# DNA Sequence Toolkit using Python

This project contains beginner-friendly Python scripts for basic DNA sequence analysis.

## Project Aim

The aim of this project is to practice Python programming while learning core bioinformatics sequence analysis concepts.

## Files Included

| File | Description |
|---|---|
| `dna_reverse_complement.py` | Calculates the reverse complement of a DNA sequence |
| `gc_content_calculator.py` | Calculates GC content percentage of a DNA sequence |
| `dna_to_rna.py` | Converts a DNA sequence into an RNA sequence |
| `motif_finder.py` | Finds positions of a motif in a DNA sequence |
| `kmer_counter.py` | Counts k-mers in a DNA sequence |

## Skills Practiced

- Python functions
- Strings and string methods
- Dictionaries
- Loops
- Lists
- Basic DNA sequence analysis
- Reverse complement calculation
- GC content calculation
- DNA to RNA transcription
- Motif finding
- k-mer counting

## Learning Outcome

This project helped me understand how Python can be used to perform basic DNA sequence analysis tasks used in bioinformatics.
